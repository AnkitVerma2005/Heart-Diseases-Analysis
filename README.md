# 📖 Project Overview

Heart disease is one of the leading causes of death worldwide. This project provides users with educational content about cardiovascular diseases through a modern and responsive web application developed using Flask, HTML, CSS, Bootstrap, and JavaScript.

The application is easy to use and helps users understand the importance of maintaining a healthy lifestyle to reduce the risk of heart disease.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/AnkitVerma2005/Heart-Diseases.git
```

## Move to Project Folder

```bash
cd Heart-Diseases
```

## Create Virtual Environment

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install flask
```

or

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
